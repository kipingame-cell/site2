# Матрица Судьбы — site2

Бесплатный калькулятор Матрицы Судьбы: личный расчёт, совместимость, чакры, прогноз по годам — с полной базой трактовок и классической восьмиконечной октаграммой.

## Запуск

```bash
node server.js          # http://localhost:3000
PORT=8080 node server.js
```

Нулевые зависимости — только Node.js 18+.

## Тесты

```bash
node --test tests/      # 10 тестов расчётного ядра
```

## Структура

```
server.js                 статический сервер (node:http, без зависимостей)
public/
  index.html              интерфейс
  css/style.css           премиальная тёмная тема (золото/фиолет)
  js/
    core/matrixCore.js    расчётное ядро (НЕ менять — формулы проверены)
    data/arcana.js        встроенная краткая база (fallback)
    db.js                 загрузчик полной базы трактовок
    octagram.js           отрисовка классической октаграммы (SVG)
    app.js                логика интерфейса
  db/                     полная база трактовок (импортируется из site)
tests/matrixCore.test.mjs
deploy/nginx.conf         конфиг для домена destiny_matrix
.github/workflows/import-db.yml
```

## Импорт полной базы трактовок

База (~59 файлов: личные зоны, здоровье по чакрам, совместимость, родовые программы)
живёт в приватном репозитории `site` и подтягивается автоматически:

1. Временно сделай репозиторий `site` публичным (Settings → Danger Zone → Change visibility).
2. Здесь в `site2`: вкладка **Actions → «Import DB from site repo» → Run workflow**.
3. Верни `site` в приватный режим.

До импорта сайт уже работает на встроенной краткой базе — бейдж в шапке показывает,
какая база активна.

## Деплой на домен destiny_matrix

```bash
node server.js            # или pm2 / systemd
```

Nginx проксирует на порт 3000 — готовый конфиг в `deploy/nginx.conf`.
