# Матрица Судьбы (site2)

Чистая пересборка сайта Матрицы Судьбы. Без фреймворков, без сборщика, без npm-зависимостей.

**Возможности**

- Личная матрица по дате рождения (22 аркана): портрет, таланты, материальная карма, кармический хвост, предназначения, деньги, отношения, род, матрица здоровья (7 чакр), прогноз по годам.
- Матрица совместимости для двух дат (поузловая сумма матриц).
- Весь расчёт — локально в браузере, даты никуда не отправляются.
- Интерактивная SVG-схема октаграммы: клик/наведение на точку показывает её смысл.

## Запуск

### Локально

```bash
node server.js          # http://localhost:3000
PORT=8080 node server.js # другой порт
```

Требуется только Node.js 18+. `npm install` не нужен — зависимостей нет.

Можно и вовсе без Node: папка `public/` — чистая статика, открывается любым веб-сервером
(важно: отдавать по HTTP, не через `file://`, т.к. используются ES-модули).

### Тесты расчётного ядра

```bash
npm test        # node --test tests/
```

Эталоны в тестах сверены с ручными расчётами: 03.10.1974, 25.11.2002, 08.03.1960, 07.03.1946, 19.08.1883.

## Домен destiny_matrix

**Вариант 1 — свой сервер (VPS):**

1. Склонировать репозиторий, запустить `node server.js` (через systemd/pm2).
2. Настроить nginx по примеру `deploy/nginx.conf` (прокси на :3000 или прямая раздача `public/`).
3. DNS: A-запись домена на IP сервера. HTTPS: `certbot --nginx`.

**Вариант 2 — GitHub Pages:**

Settings → Pages → Source: `main`, папка `/public` (или перенести содержимое `public/` в корень ветки `gh-pages`).
Затем Settings → Pages → Custom domain: указать домен, в DNS создать CNAME на `kipingame-cell.github.io`.

```
  css/style.css            # тема
  js/app.js                # UI, SVG-схема, секции разбора

tests/matrixCore.test.mjs  # эталонные тесты ядра
deploy/nginx.conf          # пример конфига под домен
```

## Формулы (метод 22 арканов)

Свёртка: любое число > 22 приводится суммированием цифр до диапазона 1–22.

- `A` — день рождения · `B` — месяц · `C` — сумма цифр года
- `D` (хвост) = A+B+C · `E` (центр) = A+B+C+D
- Диагонали рода: A+B, B+C, C+D, D+A
- Подточки оси: inner = угол+E, mid = угол+inner
- Ключи: вход = bottom.inner+right.inner, отношения = bottom.inner+вход, деньги = вход+right.inner
- Хвост: [E+D, (E+D)+D, D]
- Предназначения: небо = B+D, земля = A+C, личное = небо+земля;
  линия отца = (A+B)+(C+D), линия матери = (B+C)+(D+A), соц = отец+мать,
  общее = личное+соц, планетарное = общее+соц
- Здоровье: физика по линии Земли (ось день–год), энергия по линии Неба (ось месяц–хвост),
  итог чакры = физика+энергия; итоги столбцов — суммы со свёрткой
- Совместимость: сумма одноимённых точек двух матриц со свёрткой
- Прогноз года Y: день + месяц + сумма цифр Y (со свёрткой)
